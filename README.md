10개의 점수/ 평균 / 최고점수 최저점수 / 최고점수 최저점수의 차이를 계산하고 출력하는 프로그램이다

void get_data(int* p)

void print_data(int* p)

void print_SumAvg(int* p)

int get_Max(int* p)

int get_Diff(int Max, int Min)
